RailView — Train Schedule & Seat Reservation System

 A web-based application for checking train schedules and seat availability across Indian Railway routes.

Group Project — 3 Members
  This project was built as a college group project. All three members contributed equally across the development of the system.

Technologies handled by the team

1. HTML — Structure and layout of the application
2. CSS — Styling and visual design
3. JavaScript — Logic, search functionality, and interactivity

About the Project

 RailView is a Train Scheduling and Seat Reservation Checking System that allows users to:

 1. Search for trains between stations
 2. View seat availability in real time
 3. Register and log in to an account
 4. Book available seats and receive a PNR number
 5. View and cancel existing bookings

How to Run
1. Download or clone this repository
2. Open `index.html` in any web browser
3. The app runs with built-in demo data — no setup needed!
4. To connect a real backend, update the API URL inside `index.html`:
javascript const API = 'http://localhost:5000/api';
